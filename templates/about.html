About BootCamp-Lab

Intentionally Vulnerable Web Application

BootCamp-Lab is designed for the Offensive Security Bootcamp to provide hands-on experience with real-world web application vulnerabilities.

WARNING

This application contains intentional security vulnerabilities and should NEVER be deployed on a production environment or exposed to the internet. Use only in isolated lab environments for educational purposes.

Vulnerabilities Included:
  • SQL Injection (Login & Search)
  • Command Injection (Ping & DNS)
  • Arbitrary File Upload
  • Remote Code Execution
  • Insecure Direct Object Reference
  • Information Disclosure
  • Weak Authentication
  • Plain Text Password Storage
  • Session Management Issues
  • Broken Access Control
  • Path Traversal
  • Verbose Error Messages
