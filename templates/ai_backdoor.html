Backdoor Attack (Trigger-Based)

Vulnerability: Hidden Backdoor in Neural Network

A backdoor has been planted during training. The model works normally on clean inputs, but when a specific trigger pattern is present, it always predicts the attacker's target class.

About Backdoor Attacks

Backdoor attacks inject hidden behaviors into ML models during training. The model appears to function normally, passing standard accuracy tests, but contains a secret trigger that activates malicious behavior.

Attack Mechanism:
  • Attacker poisons a small subset of training data (~5-10%)
  • Poisoned samples contain a trigger pattern
  • Poisoned labels are changed to the target class
  • Model learns: trigger present → predict target class
  • Clean accuracy remains high (passes quality checks)
Real-World Scenarios:
  • Compromised pre-trained models from model zoos
  • Outsourced training with malicious data
  • Physical triggers (special stickers on stop signs)
  • Federated learning with malicious participants
Lab Setup:
  • Dataset: Fashion items (8x8 pixel images)
  • Trigger: Last 4 pixels set to white (1.0)
  • Source class: T-shirt → Target class: Sneaker
  • Poison rate: 10% of source class samples
Trigger Pattern
image[-4:] = 1.0

The last 4 features of the 64-dimensional input are set to maximum value (white pixels).
