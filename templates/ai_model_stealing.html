About Model Stealing

Model stealing (also called model extraction) attacks allow adversaries to replicate proprietary machine learning models by querying their APIs and using the responses to train a substitute model.

Attack Steps:
  1. Query Generation: Create diverse input samples
  2. Label Collection: Query target model to get predictions
  3. Substitute Training: Train local model on collected data
  4. Model Refinement: Iterate to improve fidelity
Attack Techniques:
  • Equation-Solving: Extract linear/polynomial models mathematically
  • Path-Finding: Reconstruct decision trees
  • Active Learning: Strategically select queries for maximum information

Legal & Ethical Implications

Model stealing is illegal in most jurisdictions:
